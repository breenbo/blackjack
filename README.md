# BlackJack game

## Features :
   - 1 to 4 players
   - random player's and dealer's card
   - dealer hit while his score <17
   - hit, stand, double or quit
   - bets can be picked
   - manage amount of player's money, with bonuses for double, split and blackjack
   - use OOP
   - lots of fun
